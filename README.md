# Files

## `settings.json`
VS Code LaTeX Workshop 配置文件 特点：
1. 输出pdf和tex在同一文件夹，其它中间文件在build文件夹且不自动清除（方便debug且当前目录不会被中间文件强碱）
2. 有用latexmk编译XeLaTeX多次、pdfLaTeX的选项，也有单独的编译选项
3. 未启用 ChkTeX （中文环境容易报大量误判warning）

## `ZDDJQ1227.apk`
自动点击器无广告版（来自Telegram群）

## `proxywsl.sh`
让WSL走Windows Clash代理的一种可行配置（本人目前没用这个配置，更推荐用 `~\.wslconfig`）
